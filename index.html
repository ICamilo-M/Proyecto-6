Proyecto extra: App Gestión atrasos

Este es un proyecto personal externo al contenido del bootcamp, realizado con framework Laravel y base de datos Mysql.
Se trata de una aplicación web que permite la gestión y control de ingresos de alumnos atrasados a un establecimiento educacional, así como también dentro de éste, a la sala de clases.

Cuenta con funciones automatizadas de notificaciones y citaciones vía correo electrónico a los padres y apoderados (representante del alumno), así como funciones de usabilidad y exportación de datos para su análisis y toma de desiciones. Además permite su uso de manera semi automatizada por medio de la lectura de código de barra.

Esta aplicación web a sido ideada y diseñada en funcionalidades, modelo de negocios y definición de uso de tecnología PHP, Bases de datos relacionales SQL y Framework para PHP. la programación por demanda de clientes, a sido realizada por terceros colaboradores que fueron contratados para el proyecto.
Actualmente se encuentra en producción y a producido en impacto positivo en la mejora del cumplimiento de responsabillidades para con los educandos.

Esta página de inicio ha sido construida empleando la tecnología Bootstrap (con un tema creado por Start Bootstrap). En su estructura se encuentra el código fuente en HTML, CSS y JavaScript, ofreciendo también la opción de integrar hojas de estilo SASS para facilitar una personalización más detallada.

Este portafolio representa un hito significativo al ser el pionero en una serie de implementaciones estratégicas. Su principal objetivo es destacar y presentar de manera integral las actividades de prácticas de programación realizadas a lo largo del bootcamp de developer full stack, una experiencia formativa respaldada y patrocinada por UCamp. Al ser la primera iteración de esta serie de portafolios, se ha cuidado meticulosamente cada aspecto para reflejar no solo el desarrollo técnico adquirido durante el programa, sino también para proporcionar una visión completa y clara de las habilidades y logros alcanzados. La elección de tecnologías específicas, la estructura del código, y la presentación visual han sido cuidadosamente consideradas para transmitir de manera efectiva el nivel de competencia y el compromiso con la excelencia en el desarrollo full stack. Este portafolio no solo sirve como un escaparate de habilidades técnicas, sino también como una narrativa que subraya el aprendizaje continuo, la adaptabilidad y la dedicación para sobresalir en el ámbito del desarrollo de software. Al destacar las prácticas de programación específicas y los proyectos realizados, este portafolio se convierte en una herramienta efectiva para comunicar la capacidad de aplicar conocimientos teóricos en situaciones del mundo real, una habilidad invaluable en el campo de desarrollo full stack.
